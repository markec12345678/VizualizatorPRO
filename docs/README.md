## 🔄 Vzdrževanje dokumentacije

Dokumentacija se posodablja ob vsakem release-u. Pravila:

1. **README.md** je vedno aktualen z najnovejšimi funkcionalnostmi
2. **CHANGELOG.md** se posodobi ob vsakem PR-ju
3. **API.md** se posodobi ko se doda/spremeni API endpoint
4. **DATABASE.md** se posodobi ko se spremeni Prisma shema
5. **ARCHITECTURE.md** se posodobi pri večjih arhitekturnih spremembah
